You can use the following tags in your formats:
{name} - Name of the theme (based on style.css)
{desc} - Description of the theme (based on style.css)
{themeurl} - Theme URI (based on style.css)
{author} - Author name (based on style.css)
{version} - version number of the theme (based on style.css)
{folder} - Folder name of the theme
{demo} - URL to preview the theme (?themedemo=xxx)
{hits} - Total no. of previews
{screenshot} - URL to theme's screenshot file
{infopage} - URL to the theme's info page (if exist)
{downloadpage} - URL to the theme's download page (if exist)
{buypage} - URL to the page to buy the theme (if exist)
